DinkToPdf Native Library Setup
================================

To use DinkToPdf for PDF generation, you need to download the wkhtmltopdf native library.

For Windows (64-bit):
1. Download from: https://github.com/rdvojmoc/DinkToPdf/tree/master/v0.12.4/64%20bit
2. Download the file: libwkhtmltox.dll
3. Place it in this folder (wkhtmltopdf/)

For Windows (32-bit):
1. Download from: https://github.com/rdvojmoc/DinkToPdf/tree/master/v0.12.4/32%20bit
2. Download the file: libwkhtmltox.dll
3. Place it in this folder (wkhtmltopdf/)

Alternative download:
You can also download from the official wkhtmltopdf releases:
https://wkhtmltopdf.org/downloads.html

After placing the DLL file:
- Make sure the file is named: libwkhtmltox.dll
- The application will load it from this directory

Note: The DLL file is not included in the repository due to its size.
