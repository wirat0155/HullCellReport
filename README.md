# HullCellReport

## วัตถุประสงค์ของระบบ (System Objectives)

ระบบ **HullCellReport** ถูกพัฒนาขึ้นเพื่อใช้ในการบริหารจัดการและรายงานผลการวิเคราะห์คุณภาพน้ำยาชุบโลหะด้วยวิธี Hull Cell Test ระบบนี้ช่วยอำนวยความสะดวกในการบันทึกข้อมูล การวิเคราะห์ผล และการจัดเก็บข้อมูลอย่างเป็นระบบ เพื่อให้มั่นใจในคุณภาพของกระบวนการชุบโลหะ พร้อมทั้งรองรับการออกรายงานและการเชื่อมต่อข้อมูลกับระบบอื่น ๆ

## ฟีเจอร์หลัก (Main Features)

### 1. การจัดการรายงาน (Report Management)

- **สร้างรายงาน (Create Report):** บันทึกผลการทดสอบ Hull Cell พร้อมรายละเอียดต่าง ๆ ของบ่อชุบ
- **ดูและแก้ไขข้อมูล (View & Edit):** เรียกดูรายละเอียดของรายงานตาม UUID และแก้ไขข้อมูลได้
- **ตรวจสอบสถานะ (Validation):** ระบบรองรับการตรวจสอบ (Check) และวิเคราะห์ (Analyze) รายงาน โดยมีการระบุตัวตนผู้กระทำ
- **รายงานค้าง (Pending/Incomplete):** แจ้งเตือนหรือตรวจสอบรายงานที่ยังดำเนินการไม่เสร็จสิ้น

### 2. การออกรายงานและเอกสาร (Export & Documentation)

- **PDF Export:** สร้างไฟล์รายงานในรูปแบบ PDF ที่มีความสวยงามและเป็นมาตรฐาน
- **Auto Save to Server:** ระบบสามารถบันทึกไฟล์ PDF ไปยัง Network File Server (SMB) ได้โดยอัตโนมัติเพื่อการจัดเก็บแบบถาวร

### 3. การเชื่อมต่อระบบภายนอก (Integration)

- **IoT & Monitoring:** เชื่อมต่อกับ **ThingsBoard** เพื่อส่งข้อมูลสภาพแวดล้อมหรือค่าต่าง ๆ ไปยัง Dashboard กลาง
- **Automation:** รองรับการดึงข้อมูลอัตโนมัติจาก **Node-RED** (Auto Feed Data) เพื่อลดการคีย์ข้อมูลด้วยมือ

### 4. ระบบจัดการผู้ใช้และความปลอดภัย (User & Security)

- **Authentication:** ระบบล็อกอินเข้าใช้งาน
- **Permissions:** การจัดการสิทธิ์ผู้ใช้งานในการเข้าถึงเมนูและการกระทำต่าง ๆ (เช่น การอนุมัติรายงาน)
- **Employee Data:** เชื่อมโยงกับฐานข้อมูลพนักงานเพื่อดึงข้อมูลชื่อและรหัสพนักงาน

### 5. อื่น ๆ (Others)

- **Dashboard:** หน้าภาพรวมสำหรับดูสถานะล่าสุด
- **Image Handling:** รองรับการอัปโหลดและบีบอัดรูปภาพแผ่น Hull Cell เพื่อประกอบรายงาน
